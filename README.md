# Boston Abula Spot 🍲

A premium, interactive web application for **Boston Abula Spot**, specializing in hot, authentic Nigerian dishes (Amala, Pounded Yam, Semolina, Eba, Jollof Rice, rich soups, and drinks) delivered directly to your doorstep.

## 🚀 Features

- **Interactive Orbiting Hero Wheel**: A custom-designed desktop carousel showcasing dish items in an orbital carousel with a responsive layout.
- **Custom Abula Builder**: Build your plate by selecting swallows, soups, and custom protein counts with dynamic total updates.
- **Modern Shopping Cart**: Add, update, and remove menu selections with automated coupon codes.
- **Checkout & Transfer System**: Guides users through offline transfer payment with simple copy-to-clipboard details.
- **Order Tracking**: Demo timeline progress simulation alongside a persistent **Delivery Location Form** storing addresses locally.
- **Modern responsive UI/UX**: Mobile Hamburger toggle menu overlay with a floating card layout, smooth animations (Framer Motion), and HSL color theme.

## 🛠️ Tech Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Library**: [React 19](https://react.dev/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 💻 Getting Started

### Prerequisites

Ensure you have Node.js installed (version 18+ recommended).

### Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Run the development server:

   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## 📦 Building for Production

Compile the production build:

```bash
npm run build
```

Start the production server:

```bash
npm run start
```

## 🌐 Vercel Deployment

Deploy this project instantly to Vercel. Next.js works out of the box using Vercel's automated builder preset, configured in `vercel.json`.
